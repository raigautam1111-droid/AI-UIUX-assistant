# AI-UIUX-assistant
AI-powered UI/UX feedback &amp; layout generator
 AI-Powered UI/UX Assistant

An AI-powered SaaS tool that analyzes UI designs and suggests improvements or generates layout ideas from text prompts.

 Features
- Upload UI screenshots or Figma links
- AI-powered design feedback (spacing, color, hierarchy)
- Generate UI layout ideas from text prompts
- Modern, premium SaaS UI/UX

Tech Stack
Frontend
- React + Vite
- Tailwind CSS
- Framer Motion

Backend
- FastAPI
- OpenAI Vision / LLM APIs
- Computer Vision (CLIP / ViT - optional)
